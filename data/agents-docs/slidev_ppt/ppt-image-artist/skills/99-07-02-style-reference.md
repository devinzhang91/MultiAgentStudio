# 风格参考速查

快速参考两种限定风格的描述词和适用场景。

## 8bit像素风格

### 完整描述词
```
8-bit pixel art 2D style, large chunky pixel blocks, brick-style aesthetics, 
Kairosoft retro game aesthetic, 2D pixelated graphics, limited color palette, 
nostalgic gaming vibe, chunky pixels, classic arcade style, vibrant colors
```

### 视觉特征
- 大块像素点（ chunky pixels）
- 有限的调色板
- 复古游戏美学
- 2D像素化图形
- 鲜明的颜色

### 适用场景
| 场景类型 | 示例 |
|---------|------|
| 游戏相关 | 游戏机制说明、游戏行业分析 |
| 复古主题 | 历史回顾、怀旧内容 |
| 活泼氛围 | 团队介绍、趣味知识 |
| 简化概念 | 复杂技术概念的简化展示 |

### Prompt添加方式
在主体描述后追加风格词：
```
[主体描述], 8-bit pixel art 2D style, large chunky pixel blocks, 
brick-style aesthetics, Kairosoft retro game aesthetic, 2D pixelated graphics, 
limited color palette, nostalgic gaming vibe, chunky pixels, 
classic arcade style, vibrant colors
```

## 扁平矢量风格

### 完整描述词
```
flat design illustration, vector art style, thick bold black outlines, 
simple rounded shapes, infographic aesthetic, educational explainer video style, 
solid flat colors, no gradients, minimal shading, clean light background
```

### 视觉特征
- 粗黑轮廓线
- 简洁圆润的形状
- 信息图表美学
- 纯色填充（无渐变）
- 最少阴影
- 干净浅色背景

### 适用场景
| 场景类型 | 示例 |
|---------|------|
| 商务演示 | 商业计划、市场分析 |
| 教育科普 | 知识讲解、概念说明 |
| 流程图表 | 工作流程、系统架构 |
| 专业报告 | 数据分析、年度总结 |

### Prompt添加方式
在主体描述后追加风格词：
```
[主体描述], flat design illustration, vector art style, 
thick bold black outlines, simple rounded shapes, infographic aesthetic, 
educational explainer video style, solid flat colors, no gradients, 
minimal shading, clean light background
```
